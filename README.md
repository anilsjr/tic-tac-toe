## Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/anilsjr/tic-tac-toe.git
    ```
2. Navigate to the project directory:
    ```sh
    cd tic-tac-toe
    ```

## Usage

1. in your web browser start the game and enjoy.
